### Section 2 — Why Windows Can't Open HEIC by Default

**要点（只写概览）：**
- HEIC 是 Apple 的高效图像格式，iOS 默认使用
- Windows 没有原生授权，需要额外组件
- Microsoft 提供官方扩展（免费/付费取决于地区）
- 解释 HEIF vs HEIC 的区别（简短）

**写法要求：** 2-3 句说明原因，不需要深入技术细节

**关键词分布：**
- `why can't windows open heic`
- `heic format windows support`

### Section 3 — Option 1: Install Microsoft HEIF Extensions

**要点（只写方法概览，不写编号步骤）：**
- 官方方案：从 Microsoft Store 安装 HEIF Image Extensions
- 安装后可在 Photos app 和 File Explorer 预览 HEIC
- 注意：部分地区可能需要付费（约 $0.99）
- 局限：只能查看，不能编辑或转换

**写法要求：** 2-3 句概览，说明优缺点

**关键词分布：**

**注意：** 提供 Microsoft Store 链接
